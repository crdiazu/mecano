# **App Name**: Mecano Solutions Website

## Core Features:

- Product Showcase: Displays Mecano Solutions' range of metal shelving and storage solutions with images and descriptions.
- Contact Form: A form for potential customers to request a quote or ask questions.
- Company Information: Provides details about Mecano Solutions, including its certifications (ISO 9001:2015, NCH2369) and areas of service.
- Parallax Scrolling Effect: Adds a parallax scrolling effect to background images for a visually appealing user experience.
- Mobile-Responsive Design: Ensures the website is fully responsive and adapts to different screen sizes and devices.

## Style Guidelines:

- Primary color: Deep orange (#F28C38), reflecting the brand's energy and focus on solutions. 
- Background color: Dark gray (#2D2D2D) creates a professional, modern backdrop, setting off the content in an elegant way.
- Accent color: Soft amber (#F59E0B), analogous to the primary color but distinct, used to highlight interactive elements and important information.
- Headline font: 'Montserrat' sans-serif for headlines. Display a strong, industrial style that is easy to read, well-suited for drawing attention and providing clarity in titles. 
- Body font: 'Inter' sans-serif for body text. Modern, neutral, highly legible in various sizes and weights.
- Use sharp, clean icons relevant to shelving, storage, and industry to reinforce visual understanding. The icons must be designed with line-art style, for maximum legibility at different scales. The icons' stroke color should be near-white.
- Subtle fade-in animations on scroll to create a dynamic and engaging user experience.
